1. Что такое указатель в языке C++?
a) Переменная, которая хранит значение
b) Переменная, которая хранит адрес другой переменной
c) Одна из математических операций
d) Функция для работы с памятью

Ответ: b) Переменная, которая хранит адрес другой переменной

2. Что такое оператор разыменования в языке C++?
a) Оператор, который приводит тип переменной
b) Оператор, который возвращает адрес переменной
c) Оператор, который возвращает значение по адресу
d) Оператор, который связывает переменные

Ответ: c) Оператор, который возвращает значение по адресу

3. Как объявить указатель на целочисленную переменную в языке C++?
a) int *p;
b) int p;
c) *int p;
d) int &p;

Ответ: a) int *p;

4. Как получить адрес переменной в языке C++?
a) С помощью оператора +
b) С помощью оператора -
c) С помощью оператора &
d) С помощью оператора *

Ответ: c) С помощью оператора &

5. Какой тип имеет указатель на переменную типа double в языке C++?
a) double*
b) int*
c) char*
d) float*

Ответ: a) double*

6. Какой тип имеет указатель на указатель в языке C++?
a) int*
b) double**
c) char***
d) float*

Ответ: b) double**

7. Что произойдет, если попытаться получить значение по нулевому указателю в языке C++?
a) Программа завершится с ошибкой
b) Будет получено значение 0
c) Будет получено случайное значение
d) Ничего не произойдет

Ответ: a) Программа завершится с ошибкой

8. Какой оператор используется для выделения памяти под новый объект в языке C++?
a) new
b) malloc
c) alloc
d) create

Ответ: a) new

9. Какой оператор используется для освобождения выделенной памяти в языке C++?
a) delete
b) free
c) dealloc
d) remove

Ответ: a) delete

10. Какая функция используется для сравнения двух указателей в языке C++?
a) cmp
b) compare
c) memcmp
d) ==

Ответ: d) ==

11. Какой оператор используется для получения адреса элемента массива в языке C++?
a) &
b) *
c) []
d) .

Ответ: c) []

12. Какой оператор используется для доступа к члену структуры через указатель в языке C++?
a) &
b) *
c) []
d) ->

Ответ: d) ->

13. Что произойдет, если попытаться обратиться к невыделенной памяти в языке C++?
a) Программа завершится с ошибкой
b) Будет получено значение 0
c) Будет получено случайное значение
d) Ничего не произойдет

Ответ: a) Программа завершится с ошибкой

14. Как можно передать указатель на функцию в качестве аргумента в другую функцию в языке C++?
a) С помощью оператора &
b) С помощью оператора *
c) С помощью typedef
d) Прямо как обычную переменную

Ответ: d) Прямо как обычную переменную

15. Что такое null-указатель в языке C++?
a) Указатель на нижний предел диапазона типа данных
b) Указатель на верхний предел диапазона типа данных
c) Указатель на пустоту
d) Указатель на случайное значение

Ответ: c) Указатель на пустоту
